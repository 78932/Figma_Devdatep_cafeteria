# ☕ Mosaic – Coffee Landing Page

## 🌿 Descripción

Mosaic es una propuesta de diseño web para una cafetería moderna, enfocada en transmitir calidez, elegancia y una experiencia agradable para el usuario.

El diseño busca recrear el ambiente acogedor de una cafetería a través de una interfaz visual atractiva, combinando imágenes, colores naturales y una estructura clara.

---

## 🎯 Objetivo

Crear una landing page que:

- Transmita la esencia de una cafetería  
- Sea visualmente atractiva  
- Ofrezca una navegación sencilla  
- Mejore la experiencia del usuario  

---

## 🎨 Diseño

El proyecto está basado en:

- Colores cálidos (marrones, beige y verde)  
- Tipografía elegante  
- Imágenes realistas de café  
- Diseño limpio y organizado  

---

## 🧩 Secciones principales

- Inicio  
- Descubre el café  
- Menú  
- Nuestra esencia  
- Ambiente  
- Suscripción  
- Footer  

---

## 💡 Enfoque UX/UI

Este proyecto aplica principios como:

- Jerarquía visual  
- Uso correcto de espacios  
- Botones claros (call to action)  
- Navegación intuitiva
